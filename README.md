# Quiz Game (OOP Python)

A simple True / False quiz game built with Python using Object-Oriented Programming.

This project was created as part of my Python learning journey while following the "100 Days of Code" course by Angela Yu.

The project was not built 100% independently. I used guidance and explanations while learning the logic and structure behind OOP concepts.

---

## Features

- True / False quiz system
- Question objects using classes
- Quiz manager class (`QuizBrain`)
- Score tracking
- Randomized questions

---

## What I practiced

- Classes and objects
- `__init__`
- Object lists
- Methods
- OOP structure
- Loops and conditionals
- Working with external data files
- Basic game flow logic
